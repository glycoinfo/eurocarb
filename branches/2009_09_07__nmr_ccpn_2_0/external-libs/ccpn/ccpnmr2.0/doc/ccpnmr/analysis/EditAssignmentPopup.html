This popup is used to specify or alter which resonances are assigned to a chosen peak. One or more resonances may be connected to each dimension of the peak. The peak currently represented in the popup is the one closest to the mouse cursor when the popup was activated. The popup is activated either by pressing the "a" key or by selecting the Assign option from the Peak sub-menu of the window menu (opened with a right mouse click).

If the popup is opened from the main Analysis menu (Assignment::AssignmentPanel) the displayed peak is the last peak for which the popup was activated (or none).

Structure: <list>

This menu sets which structure is used for measuring H-H distances, and displaying peak connections or assignment possibilities (according to the current ststus of the popup).

- Assignment possibilities -

The rows of tables in this section correspond to the dimensions of the current peak. The left hand tables display the positions and connected resonances for each of these dimensions. The right hand tables display the resonances to which the peak dimension could possibly be assigned. The resonances present in these tables are determined by the assignment tolerances (set via EditSpectrumPopup. and any options selected below. There is also always the possibility of using a new resonance, which will be created when the peak connection is set.

Clicking on one of the resonance possibilities connects the resonance to the peak dimension and the left hand table is updated accordingly. If a current resonance is selected in the left hand table and a possibility in the right hand table is clicked, the selected resonance is replaced with the new option. If there is no currently selected resonance in the left hand table the resonance from the right hand table will be added to the peak dimension.

Multiple resonances assigned to a peak dimension indicate that the peak is ambiguously assigned or assigned to both/all resonances. This is common where a peak is due to the resonances of two sets of atoms with unresolvable chemical shifts. For example, unresolvable lysine delta hydrogens (Hd*) means the connection to both Hd2 and Hd3 atoms.

By default the resonances in the right hand table are ordered in terms of difference in chemical shift to the position of the peak dimension (delta). However if a structure (preliminary or otherwise) is active within the current project, hydrogen resonance possibilities may also be sorted in to order of distance (of any assigned atoms) given the structure. The distance, if displayed, is that between the resonance option and any resonance assigned to the next hydrogen dimension. The average position is used if multiple resonances are assigned to a dimension.

- Options -

These options set which resonance possibilities are displayed in the right hand tables.

<> Peak may be aliased?

With this option set to ON the list of possible resonances (right hand table) includes those that may be assignable to the peak dimension (within tolerances) only if the position of the peak in this dimension were unaliased (moved an integral number of spectral widths). Such resonance options are coloured yellow in the right hand table.

If a peak dimension is then connected to one of these resonances its aliasing will be appropriately set, i.e. its position will be unaliased so that the position of the peak closely matches that of the assigned resonance.

<> Restrict to repeated spin systems?

With this option set to ON the list of possible resonances for a dimension may be restricted to those that have spin systems which appear in at least two dimensions

Note that no restrictions will be applied to resonances that are not associated with any spin system. Also, if an entire dimension has no resonance possibility with a repeated spin ststem, then the dimension will not be subjected to restrictions. The idea here is that it is better to leave the resonance possibilites unrestricted rather than have a dimension with no possibilites.

<> Bound resonances

With this option set to ON the choice of resonances available in the right hand tables is restricted to show only those resonances that are assigned to an atom (or atoms) that are directly bonded to another atomic assigned resonance in the list of possibities.

For example this option allows the user to narrow many resonance possibilities for each dimension down to justs a few directly bound resonances. For example when assigning an 15N-HSQC-TOCSY the resonances may be limited to only connected amide proton and nitrogens.

<> Double tolerances

With this option set to ON the peak dimension assignment tolerances are temporarily doubled for the purposes of increasing the number of resonance possibilities (within tolerances) displayed for each of the peak dimensions.

[ Assign <resonanceName> ] / [Re-assign <resonaonceName>]

Assigns the currently selected resonance (from the left hand tables) to a specified set of atoms. If the resonance already carries atom assignment this will be replaced by the new selection. This option opens BrowseAtomsPopup select the atoms to which the resonance will be assigned.

Before an atom set is chosen this button becomes [ Cancel assign ] and may be used to exit form the assignment process.

[ Set atom type ]

Sets the atom type (and only this) of the currently selected peak-connected resonance (left hand table). This option opens BrowseAtomsPopup to select the atom type for the resonance. Here the atom type may be selected by selecting any atom from any molecular chain of the desired type.

[ Deassign <resonanceName> ]

This allows a selected resonance (from a peak dimension) to be cleared of any atomic or spin system assignments. If the user confirms to clear atomic assignments the user will then be prompted to clear the spin system assignments. In this way a resonance can be disconnected form an atom but left in a given residue of spin system.

[ Clear contrib ]

Removes the currently selected resonance (in the left hand tables) from its peak dimension. The assignment of the resonance to any atoms remains.

[ Show On Structure ]

Highlights any atoms assigned to the current peak on the currently selected structure. If the peak has hydrogen atom assignments in two dimensions then this will be shown as a connection on the structure. If there is one hydrogen dimension fully assigned then all of the atomic assignment possibilites for any unassigned hydrogen dimension will be displyed as connections. Such connections will be coloured according to distance: <5.5 A; green, 5.5-8.0 A; yellow, >8.0 A; red.

[ Merge <dimension> Resonances ]

This option will merge all of the resonances assigned to the peak dimension of the currently selected resonance into one single resonance.

Any resonances that the user does not wish to merge may be removed from the peak dimension using the [ Clear Contrib ] option, and then re-assigned after the merge.

[ Show All Peaks ]

This option uses SelectedPeaksPopup to display a list of peaks within the current project that are assigned to any of the resonances assigned to the peak curretnl;y considered within the EditAssignmentPopup.
