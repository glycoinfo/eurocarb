This table shows the inter-residue links a template residue makes. Such links include polymer main chain connections (e.g. peptide bonds) and side chain connections (e.g. disulphide links).

Molecule: <list>

Allows the choice of a molecule from the current project for which template residues may be chosen.

Residue Type: <list>

Selects which residue type, as determined by CcpCode, the source residue has. If many residues are present for the set type, then the specific residue can be chosen in the menu to the right. Note that this two-level selection of residue type, then particular residue is necessary because a single list of all residues in a molecule may be larger than most screen sizes allow.

Source Residue: <list>

Allows the user to select which template residue of the specified type the table should display links/connections for.

| From Link |

The type of link (often prev or next in a polymer) emanating from the current residue.

| To Link |

The type of link the connected residue is in receipt of.

| To Residue |

The name of a connected residue.
