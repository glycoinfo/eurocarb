This popup is used to create molecule systems which contain all of the atoms, albeit organised into residues and chains, that NMR resonances may be assigned to. Here the user builds molecular chains based upon molecular templates created with EditMoleculePopup.

A chain has one or more chain fragments, which is a sub-division that divides its residues into what may be distinct polymer regions. For example a protein may comprise of separate disulphide bonded polymer sections, e.g. insulin.

- Molecular Systems -

This table lists the molecular systems present in the current project. Each molecular system describes a group of molecular chains which may all be assigned to. A molecular system may correspond to all of the assignable molecules in the NMR tube, or a collection of associated chains like a complex.

In most circumstances the resonances identified in a specific NMR experiment only derive from atoms in one molecular system.

[ Add Mol System ]

Creates a new, empty molecular system within the current project to which chains and residues may be added.

[ Delete Mol System]

Delete the currently selected molecular system if no NMR assignments have been made to it.

- Molecular chains -

[ Delete Chain ]

Deletes the currently selected chain if it has not been used for any assignments.

[ Copy Chain ]

Makes a copy of the selected chain in the current molecular system. This option may be used to create oligomers where there are one or more chains with the same sequence. This is the same as creating a new chain using the same molecular template as the existing chain.

[ Make chain from template: ]

Makes a new NMR assignable chain in the current molecular system using the template specified in the pulldown menu to the right.

Molecule template: <list>

Allows the user to specify a molecule from the current project which will be used as a template to create a new chain within the selected molecular system.

[ Edit Molecule Templates ]

Opens a popup to allow the user to create and edit molecule templates which may then be used to create chains for NMR assignable residues. Opens EditMoleculePopup. Note that templates that have already been used to create a chain may not be altered.

- Chain Fragments -

List all of the fragments (polymer subdivisions) in the currently selected molecular chain. Each fragment comprises a section of polymer connected residues, which may be bonded together in a user specified way to describe the molecule as a whole.

| Start Seq Number |

Double clicking this column allows the user to set the sequence start number for a chain fragment (the residues will be renumbered sequentially). The numbering may not overlap with any other chain fragments.

This renumbering involves setting the SeqCode value for the fragments residues. This does not affect the assignment to these residues. The connection of atoms to resonances and peaks remains unaltered, but the Analysis display will be updated to reflect the new residue label.
