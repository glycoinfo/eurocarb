This popup allows the user to set/change the reference experiments that are linked to the NMR experiments within the current project. The user can also change the dimension mapping to the reference data. Setting these links specifies what class of NMR experiment was performed. The reference experiment is described by the NMR Experiment Prototype system, and thus describes the magnetisation transfer pathway(s), atom site types, selectivity, and measurements involved with the NMR experiment.

In this way the user formalises what their "CBCACONH" experiment is, and how it is different form other NMR experiments. Using this example a 3D "CBCACONH" may be linked to a reference experiment with the unique name "h{CA|Cca}coNH". This reference experiment shows that three shift measurements are made for 1) Alpha carbons OR carbons preceding alpha carbons, 2) Amide nitrogen, 3) Amide proton. The lower case letters in the reference experiment name indicate that the nuclei were involved in the magnetisation transfer, but do not have measures shifts. The "{x|y}" nomenclature specified an optional pathway.

To give some common examples:

  • A 2D 15N HSQC would be linked to the "H[N]" reference experiment where there are two measured shifts (amide proton, amide nitrogen) and the "[]" brackets indicate that the experiment is an out-and-back experiment, i.e. Magnetisation returns to the H
  • A 2D proton NOESY would be linked to "H_H.NOESY". Here the "_" indicates that the transfer between two H nuclei is not a regular J-transfer and the ".NOESY" indicates the transfer is an NOE type.
  • Combining the above a 15N HSQC-NOESY would usually be "H[N]_H.NOESY", but the alternatives "h[N]_H.NOESY" (2D experiment) and "H_H[N].NOESY" are also possible.
  • Likewise an HNCOCA could be "H[N[co[CA]]]" (common 3D), "H[c[CO[CA]]]" (3D), "H[N[CO[CA]]]" (4D) or "H[n[cO[CA]]]" (2D).

Experiment Classification

This table allows the user to link existing NMR experiment to the reference experiments in the NmrExpPrototype package by choosing the appropriate reference experiment form a series of pulldown menus within the table.

| Name |

The name of the NMR experiment as it appears generally in Analysis. This is the user-defined name and not necessarily anything to do with the type of experiment.

| Category |

This is the general categorisation of the reference experiment, e.g. through-space or through bond. Although a given experiment ought perhaps be in multiple categories it will only be in the obvious one. So an 15N HSQC-NOESY will be "through-space" even though the HSQC part of it is through-bond. In essence this option serves to help navigation through the many available reference types. Setting this column (double click) when the reference experiment is not fully set only serves to limit the subsequent possibilities.

| Type Synonym |

This is the synonym/common name for the NMR experiment prototype form which a reference experiment is chosen. Setting this for an experiment also sets a default reference experiment, but this may be further refined in the next column.

| Full Type |

The full reference experiment name that describes the transfer pathway. Setting this (by double clicking) will fill in the other columns if they are not set. Setting the other columns first restricts the choices in this column.

Reference Dimensions

This table displays the mapping of experimental dimensions to reference experiment dimensions. This mapping will automatically be made (or rather guessed at) when the reference experiment is set in the above table. If the default is wrong the user can change the mapping. Note that an experimental dimension may be split into two or more components (ExpDimRefs) when the dimension represents multiple measurement types, e.g. a combined 15N and 13C chemical shift.

| Exp Dim |

The dimension number of the NMR experiment

| Ref Ex Dim |

The dimension number of the reference. This number can be swapped around (double click) to change the reference dimension mapping.

| Isotope |

The isotope associated with the experiment dimension.

| Ref Measurement |

The measurement in the reference experiment associated with this dimension. This column lists the step numbers of the transfer pathway that goes with the measurement, the measurement type (e.g. Shift), the type of atom site (e.g. "CA") and various other data (e.g whether constant time).
