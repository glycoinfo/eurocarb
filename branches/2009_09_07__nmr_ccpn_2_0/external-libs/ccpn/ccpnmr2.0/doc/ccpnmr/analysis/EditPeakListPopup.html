This popup lists all of the peaks within a specified peak list and allows the user to perform various functions on the peaks.

Spectrum: <list>

Allows selection of the experiment and resulting spectrum from which a peak list may then be selected. The options in the pulldown list are in the form experimentName:spectrumName.

Peak List: <list>

A list of the peak lists within the currently selected spectrum, from which a peak list is chosen to display peaks for.

Structure: <list>

Chooses a structure from the current project form which H-H distances are taken. Such distances will only be displayed if the structure is not set to <None> (this can be selected at any time) and if the current spectrum has two hydrogen dimensions.

Status: <list>

This allows the user to restrict the display of lists from the selected list to those of a stated assignment status. The default is to display all peak in a list (option is "any").

[ Make Strips ]

This option uses the selected peaks to create strips (max 15) at peak positions in the target window. The strip y positions will be the average y position for the selected peaks.

[ Find Peak ]

This option brings the selected/primary peak into view within the spectrum window selected in the adjacent pulldown list. The position of this window is centred upon the coordinates of the peak position.

Window: <list>

This pulldown list allows the user to set which spectrum window selected peaks are be displayed within (using the [ Find Peak Button ] button).

| Shift F1 | Shift F2 |...

These columns display a peak's position within each of its dimensions in units appropriate to the dimension. Double clicking these columns allows the user to edit the position of the peak. This move will be reflected within the spectrum windows.

| Assign. F1 | Assign. F2 |...

Labels describing the assignment of the peak in each of its dimensions. Double clicking these columns activates EditAssignmentPopup so that the user can change the assignment of a peak in each of its dimensions.

| H-H dist. |

This column will only be visible if a current structure is selected and the current spectrum has two hydrogen dimensions. The values in the column are inter-atomic distances in the current structure if a peak is assigned to hydrogen atom sets in two if its dimensions. The distances will be for the average position of the atoms in an atom set (e.g. in the centre of a methyl).

| Fit Method |

The name of the fitting function that was used when the position and height of the peak was determined from spectral data.

| Vol. Method |

The name of method used to calculate the volume of the peak. This is separate form the method used to find the peak location.

[ Add Peak ]

Adds a new peak to the current peak list. Uses EditPeakPopup which allows the user to manually set peak position, intensity etc.

[ Edit Peak ]

Opens EditPeakPopup so that various parameters of the selected/primary peak may be altered by the user in a separate popup.

[ Unalias Peak ]

Opens EditPeakAliasingPopup for the currently selected/primary peak. From here the user can move a peak to an unaliased position in the spectrum representative of its resonance.

[ Delete Peaks ]

Deletes the currently selected peaks. Any resonances to which they are connected are otherwise unaltered.

[ Assign Peak ]

Assign the selected/primary peak using EditAssignmentPopup.

[ Recalculate Intensities ]

Calculates the intensity values (height, volume) of the currently selected peaks based upon their present position within the spectral data. This option is used when a peak has been moved or the method of determining its intensity has been changed.

[ Show On Structure ]

Display the assigned atoms and any H-H connections corresponding to the selected peaks (in the peak list table) on the current structure. Uses ViewStructurePopup.

[ Show All On Structure ]

Display the assigned atoms and any H-H connections corresponding to all of the peak in the current peak list on the current structure. This option may take a significant time to render if the peak list is large. Uses ViewStructurePopup.
