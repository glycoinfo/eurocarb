Assign peak

Displays the possible assignment options for a peak using EditAssignmentPopup. The information is displayed for the peak that was last selected, either via the Peaks:Assignment option of the right mouse menu or by editing the assignment in one of the peak editing tables such as SelectedPeaksPopup or EditPeakListPopup.

The assignment panel allows the connection of resonances, displayed in a ranked list, to each of a peaks dimensions. Resonances may also be removed and replaced. The popup also allows the assignment of a resonance to a given set of atoms (the atoms are chosen in BrowseAtomsPopup.

Propagate assignments

Spreads the assignments, where possible and within assignment tolerances, across all of the selected peaks.

Any resonances assigned to the selected peaks will be assigned to any other valid peak dimension within the selection. Useful for quickly assigning multiple peaks which, for example, derive from the same 15N ans 1H amide resonances.

Add to spin system

Adds the resonances assigned to the currently selected peaks to a spin system. If the peaks have no spin system information a new spin system will be created. If only one spin system is represented by the peak assignments, all of the resonances assigned to the peaks will be put into this spin system. If multiple spin systems are represented these may be merged together or the operation canceled.

Predict Spin System Type

Used to predict the residue type of the spin system that the selected peak may be assigned to (via resonances). If many spin systems are assigned only the first, according to the order of the peak dimensions, is considered. Opens SpinSystemTypeScoresPopup.

Deassign selected peaks

Removes all resonance assignments from the currently selected peaks

Set sequential spin systems

Allows the user to specify the seqential relationship of the spin systems to which any resonances assigned to the selected peak belong. For example the user may choose the option 

F1 0, F2 -1, F3 0
to set the spin systems for an HN(CO)Ca peak where the 13C resonance is known to be in the previous (i-1) residue to the 1H and 15N resonances (i).
