Here the user can copy assignments between peaks in related peak lists/spectra. For example assignments can be copied between different types of spectra which share some peak positions, e.g. TOCSY and NOESY, and also between the same type of spectra under slighly different conditions where peaks may have moved.

- Options -

Source Peak List: <list>

The option allows selection of the peak list containing assigned peaks. These assigned peaks act as reference points to assign peaks in the target list.

Target Peak List: <list>

This selects the peak list containing the unassigned peaks that are to be assigned by matching their position to the source peaks.

Follow Peaks? <check_button>

Setting this option to ON means that when a source peak is selected in the table is position is centred and highlighed in an appropriate window (set below).

Restrict Peaks: <check_button>

With this option set to ON the list of source peaks is restricted so that only source peaks with at least one good positional match, in the target list, are listed. Setting the option to OFF displays information for all of the assigned peaks in the source peak list.

Follow Window: <list>

This option allows the user to set which window source peak positions are highlighed in, if the Follow Peaks option (above) is set to ON.

Dist. Threshold: <text_entry>

This entry allows setting of the minimum distance between a source and target peak positions for them to be considered (potentially) matched. The distance is calculated as the square root of the sum of the square differences in positions along each axis, where the difference for each axis location is first divided by the assignment tolerance for that dimension.

- Source Peaks -

This table contains a list of peaks which assignments may be copied from. It also indicate the degree of and number of macthes to the target peaks.

| Assignment |

The current assignment of the source peak. This acts as an identifier.

| Num. Matches |

The number of peaks found in the target peak list which have a positional match to the source peak that is within the distance tolerance.

| Best Dist. |

The shortest measured distance between the source peak and a peak of the target list, if a peak is found within tolerances. If there is no macthing target peak within tolerances this column is left blank.

| Best Match |

The assignment of the best macthing target peak that is within the distance tolerances.

- Target Peaks -

When a source peak is selected in the Source Peaks table above, this table displays a list of peaks from the target peak list that are within distance tolerances of the source. The assignments of the source peak may then be transferred to any of these matched peaks using the [ Assignn Selected Target ] button.

| Distance |

The weighted positional distance between the source peak position and the position of the target peak. The distance is calculated as the square root of the sum of the square differences in positions along each axis, where the difference for each axis location is first divided by the assignment tolerance for that dimension.

[ Assign Selected Target ]

Clicking this button causes the currently selected target peak (lower table) to receive the same assignments as the selected source peak (upper table).

[ Assign All Below Threshold ]

This button allows the user to assign all of the best matching target peaks, that residue within the distance threshold, according to their source peaks.
