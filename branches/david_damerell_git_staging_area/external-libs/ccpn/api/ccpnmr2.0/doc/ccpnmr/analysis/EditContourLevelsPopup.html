Allows the user to change the contour levels for a chosen spectrum. (The contour levels are independent of window.)

- Spectrum -

This panel specifies the experiment to set contour levels for. Note that the global scale (set to the right) affects all spectra.

Experiment: <pulldown_menu> Spectrum: <pulldown_menu>

The name of the chosen spectrum and its experiment.

[ Spectrum Properties ]:

Brings up the EditSpectrumPopup dialogue for the chosen spectrum.

- Global scale -

This panel sets the scale by which all contour levels for all spectra are multiplied before doing the actual contouring.

<text_entry>

Where the user can enter a vaule for the global scale.

[ /2 ]:

Divide the global scale by 2.

[ *2 ]:

Multiply the global scale by 2.

[ 10^5 ]:

Sets the global scale to the default of 100,000

[ ----++++ ]:

By selecting a spot on this button the global scale is continuously increased (over a +) or decreased (over a -), with the speed of change depending on how far the mouse is from the centre point.

- Auto contour levels -

In this panel the user can generate many contour levls at regular intervals by setting various simple parameters.

Base level: <text_entry>

This is the base level used to calculate the contour levels for a spectrum (before they are multiplied by the global scale).

[ /2 ]:

Divide the base level by 2.

[ *2 ]:

Multiply the base level by 2.

[ ----++++ ]:

By selecting a spot on this button the base level is continuously increased (over a +) or decreased (over a -), with the speed of change depending on how far the mouse is from the centre point.

Number of levels: <text_entry>

The number of (positive or negative) contour levels.

[ -1 ]:

Decrease the number of levels by 1.

[ +1 ]:

Increase the number of levels by 1.

[ =5 ]:

Set the number of levels to 5.

[ =10 ]:

Set the number of levels to 10.

[ =15 ]:

Set the number of levels to 15.

Level multiplier: <text_entry>

The multiplier from one contour level to the next (starting from the base level).

[ 1.3 ]:

Set the level multiplier to 1.3.

[ 1.4 ]:

Set the level multiplier to 1.4.

[ 1.5 ]:

Set the level multiplier to 1.5.

[ 1.6 ]:

Set the level multiplier to 1.6.

[ 1.7 ]:

Set the level multiplier to 1.7.

<>positive contours:

If selected the positive contours are included in the contour levels.

<>negative contours:

If selected the negative contours are included in the contour levels.

- Manual contour levels -

Here the user can specifiy individual contour levels. These levels will be automatically filled in when the above "Auto contour levels" are used.

<text_entry>

A comma separated list of the contour levels for the chosen spectrum. Note that these values will be multiplied by the global scale to give the displayed levels.

[ Apply ]

Sets the contour levels to the values shown in the contour levels text entry above.

[ Done ]

Sets the contour levels to the values shown and then closes the popup
