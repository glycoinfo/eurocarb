Displays chemical shift distributions for the atoms of the residues commonly represented in the BioMagResBank database. This popup uses ScrolledGraph to display the distributions. See separate ScrolledGraph documentation for details about configuring and exporting the graph.

The vertical axis represents the proportion of the atoms in the database (within a set tolerance). The horizontal axis represents chemical shift in ppm.

The colour of each distribution is the same as the colour of the corresponding atom option, which is selectable at the top.

Molecule Type <list>

Select the class of bio-polymer molecule to display information for. Options include Protein, DNA and RNA.

CcpCode: <list>

The CcpCodes of all the residues with database entries in for the currently selected molecular type. The CcpCode for a residue is usually the same as the regular three letter code (e.g. Asparagine is ASN).

Atom Type: <list>

Used to select whether the hydrogen atoms or other ('heavy') atoms have displayable distributions.

[ <atomName> ]

Toggled buttons to select which atoms chemical shift distributions are displayed for. When a button is coloured the corresponding atom selection is displayed. If it is not displayed the button is grey.

The toggled status (off or on) of a button will persist when the residue type is changed (only displayed if the name of the atom is appropriate).
