WindowPopup

This popup is used to group spectrum windows. Only the windows in the active group are displayed on the screen.

| Group |

The group name.

| Number of windows |

The number of windows in the group.

[ Make Active ]

Make the selected group active.

| Window |

The window name.

| In group |

Whether or not this window is in the group selected in the top table.
