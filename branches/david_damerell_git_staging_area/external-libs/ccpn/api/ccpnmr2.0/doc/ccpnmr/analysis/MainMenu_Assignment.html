Assignment Panel

Displays the possible assignment options for a peak using EditAssignmentPopup. The information is displayed for the peak that was last selected, either via the Peaks:Assignment option of the right mouse menu or by editing the assignment in one of the peak editing tables such as SelectedPeaksPopup or EditPeakListPopup.

The assignment panel allows the connection of resonances, displayed in a ranked list, to each of a peaks dimensions. Resonances may also be removed and replaced. The popup also allows the assignment of a resonance to a given set of atoms (the atoms are chosen in BrowseAtomsPopup.

Resonances

Uses BrowseResonancesPopup to display a table of all of the resonances represented in a given shift list of the current project. Resonances may be selected according to certain criteria. From the popup they can be assigned (either to atoms or spin systems), de-assigned, merged and deleted.

Spin Systems

Shows a table of all of the spin systems (Data Model: ResonanceGroups) in the current project. Uses EditSpinSystemPopup. Form here spin systems may be assigned to specific residues, assigned to residue type only, de-assigned, merged, deleted etc.

Spin System Types

This option opens SpinSystemTypingPopup to provide an interface for predicting the amino acid type of protein spin systems using the chemical shift and atom type information of any resonances contained within such a spin system. This spin system typing can be used in two ways: One is to predict the pobability of a spin system being one of the available types on an individual basis (i.e. not considering the possibilities for other spin systems), the second is to predict the spin sytem types for all of the spin systems corresponding to one protein chain. This global prediction aims to find the most likely distributions of amino acid types, form a chain, over the available spin systems. In this manner one allocation affects another because the numbers of each type of amino acid is limited in any given chain.

Assignment Graph

Uses ViewAssignmentPopup to graphically show the assignment status and connectivity for groups of atoms within a selected molecule. Here information corresponding to specified residues, atom groups and spectra may be toggled.

Assigned atom groups are circled and connections are displayed as lines between such circles. The circles and lines are colour coded according to the spectrum of origin. This display is useful for displaying the status of sequential assignment and may be used to illustrate NOE connectivity.

NOE matrix

Opens ViewNoeMatrix. This option displays the connectivity information from selected spectra, between selected molecules as a density chart. The axes of the chart correspond to the residues of the selected molecule(s). The intensity if the chart element corresponds to the number of connections between its ordinate residues.
