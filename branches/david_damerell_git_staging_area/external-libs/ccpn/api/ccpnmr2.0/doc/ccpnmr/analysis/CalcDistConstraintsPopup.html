This popup allows the generation of distance constraints form the assigned peaks within an input NOE spectrum. The values of distance for the constraint are derived form the input (or default) NOE intensity - distance table set with EditNoeClassesPopup.

- General Options -

NOSESY spectrum: <list>

Enables the selection of the input spectrum (from which a peak list is then selected) given a list of possible NOE spectra present within the current project.

Peak list: <list>

Enables the selection of the input peak list from those present for the currently selected spectra.

Bonded dimensions: <list>

Gives a list of hydrogen-heavy atom dimension pairs. The selected option specifies which dimensions of the input spectrum are considered to be directly bonded. Bonded dimensions enable the residue ranges to be applied to both bound dimensions at the same time.

Constraint Set: <list>

Determines which constraint set any generated constraints are put into. This may be an existing constraint set or a completely new constraint set. Note that the creation of a new constraint set will produce (after the shift matching routine) a separate NmrConstraints XML file for that set.

Minimum peak merit: [ float_entry ]

This value can be edited to specify the minimum merit value that a peak must posses if it is to be used to make a constraint. Setting the merit of peaks is a means of reversibly removing them from consideration.

Peak intensity type: <list>

Sets whether any distance constraints are calculated from the volume or height of the input peaks.

- Residue ranges -

Restricts the input peaks to those with assignments that lie within the specified residue sequence ranges,

Molecular system: <list>

Allows the user to select the molecular system the residue ranges apply to.

| Dimensions |

This column specifies which dimensions of the input peaks the range applies to. The presence of multiple dimensions for one range indicates that the dimensions are bonded and thus the residue range necessarily applies to both dimensions.

| Chain |

The molecular chain to which the residue range applies. This may be set by double clicking. If the chain is changes the residue range defaults to the entire length of the chain,

| Start | End |

The first and last allowable residues (according to sequence number) for the residue range. These values are always appropriate to the selected chain (otherwise they may be truncated/altered) and may be set by double clicking,

[ Add Residue Range ]

Add a new residue range to the list which may then be edited in terms of dimension, molecule chain and bounds.

[ Delete Residue Range ]

Delete the currently selected residue range. One range must be present for each set of bonded dimensions, thus the last range for a given set may not be removed.

[ Calculate constraints ]

Generate distance constraints for the assigned NOE peak in the input peak list according to the specified (or default) NOE intensity - distance class table.

[ Set distance classes ]

Opens EditNoeClassesPopup which enables the user to set a table for how the relative intensity of an NOE peak corresponds to target and limiting distances for a constraint. The relative intensity is the peak intensity divided by average intensity for the peak's peak list.
