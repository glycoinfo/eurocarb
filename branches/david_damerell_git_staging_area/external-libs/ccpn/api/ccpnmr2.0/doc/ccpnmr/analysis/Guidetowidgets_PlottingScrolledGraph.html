+/- scale - ValueRamp
The Scrolled Graph is the graphical object used to plot graphs of arbitrary data. For example it is used to plot chemical shift distributions in BrowseReferenceShiftsPopup. The functionalilty is fairly generalised, but the user can configure various options via a right mouse click activated menu.

Right Mouse Menu Options

Zoom

Allows the user to zoom in and out on the plot. Also gives the option the reset the zoom level to its default value.

Graph type

Here the user sets whether the data is plot as a scatter plot, line graph, histogram or bar chart. Note that a graph type may not be appropriate for every arbitrary data set viewed, but the user option remains nevertheless.

The main difference between the plotting of histograms and bar charts here is that the columns of a bar char are interlaced; equivalent X-axis values for different data sets lie in groups, but histogram columns may be superimposed; equivalent X-axis values are placed on top of one another.

Symbol type

Determines whether a circle, triangle or square is used to mark data points on the plot. This is only used if the graph type is set to "line" or "scatter"

Symbol size

This affects the size of the shape which is used to mark data points on the plot. This is only used if the graph type is set to "line" or "scatter"

+ increses the size one increment

- decreases the size one increment

Small/Normal/Big set the size to one of three preset values

Print to file

Allows the user to choose a file in which to save the contents of the ScrolledGraph in PostScript format. Uses FileSelectPopup.
