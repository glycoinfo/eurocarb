WindowPopup

This popup is used to control how spectrum windows are displayed. It is used to set things such as aspect ratio, background colour and axis type.

Windows: <> all windows <> active windows

Sets whether the windows listed in the table are all of those in the project or only those in the currently active window group (see edit window group popup).

- Edit Windows -

Lists the spectrum windows in the above stated category.

| Name |

The name of a window which may be altered by double clicking.

| Aspect ratio |

The ratio of the x axis scale to the y axis scale. Set by double clicking. Changing this values compresses/stretches spectrum contours displayed in the window.

| Slice range |

The height range for the 1D slices for this window. It may be set by double clicking.

| Background colour |

The colour of the background to a spectrum window. The default is white but the colour may be set by double clicking.

| Label visible |

Whether the canvas label (the strip number) is shown when there is more than one strip.

| Midpoint visible |

Whether the canvas midpoint (in the orthogonal dimensions) is shown when there is more than one strip.

[ Open ]

Open selected window (if not already open) and raise it to the front of the display.

[ Create ]

Makes a new window with the name and axes specified using NewWindowPopup.

[ Delete ]

Delete the currently selected window.

- Window axes -

Displays details about the axes of the currently selected window (top table)

| Axis Type |

The axis type, which for shift axes is the isotope type.

| Panel Type |

The panel type. This is a refinement of axis type which is used for rulers. So any axes with the same panel type have the same axis type but not necessarily vice versa. The panel type is initially determined automatically by the program. May be set with a double click.

| Scrollbar visible |

States whether a scrollbar is present for the axis within the window. Toggled between "yes" and "no" by double clicking.

| Slice visible |

Toggled between "yes" and "no" by double clicking. States whether a slice display is visible for an axis. The spectrum window will be rendered accordingly (see WindowPopup.

| Slice height |

The height in pixels of the slice display for an axis. Set by double clicking.

[ Create ]

Create a new z axis for the window. The user may then set the panel type for the axis. This causes the selected spectrum window to be redisplayed with an extra scrollbar corresponding to the new axis.

[ Delete ]

Delete the selected window axis. Note that the x or y axes of a window may not be deleted. The spectra which are mapped to the window are automatically updated.
