New Window

Opens NewWindowPopup which allows the creation of new spectrum windows with a given name and a choice of axes. The choice of other axes may replicate those of existing windows, but the name must be unique.

Edit Window

Opens EditWindowPopup. Here the user can see the attributes of all, or just the active, spectrum windows. Attributes, such as window name and aspect ratio may be edited within the tables. A second table is available for editing the individual axis attributes for the selected window.

Mapped Spectra

Uses EditWindowSpectraPopup to manipulate how spectra are mapped into the graphical windows. This option can be used to change how contours and peak lists are displayed. Also, where a spectrum has multiple axes of a given type, which spectrum dimension corresponds to which window dimension can be changed.

Window Groups

This option allows the grouping of windows using EditWindowGroupPopup. Only the currently active group will be available in the graphical interface, but the members of the group and the active group can be changed at any time.

This functionality is intended to remove clutter, especially for situations where distinctly different sets of windows are used for different purposes. For example, one group may be used for protein backbone assignment, and another for side chain assignment.

Print Window

Opens PrintWindowPopup which allows the output of spectrum windows as PostScript, Encapsulated PostScript (EPS) and Portable Document Format (PDF) files. Several style and geometry options are available.
