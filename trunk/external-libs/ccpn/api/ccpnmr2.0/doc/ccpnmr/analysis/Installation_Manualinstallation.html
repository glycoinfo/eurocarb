Analysis installation script
Manual installation
Final steps

Alternatively, if for some reason the script does not work, below is what you need to do to install the code (CCPNMR_TOP_DIR is the ccpnmr directory created by the unpacking of the tar file):

1. If you need our version of Tcl:

cd $CCPNMR_TOP_DIR/tcl8.3
Unpack tar file: 
tar xvzf tcl8.3.4.tar.gz
(or "gunzip" followed by "tar xvf" if your tar does not understand "z") Compile code: 
cd tcl8.3.4/unix
./configure --prefix=$CCPNMR_TOP_DIR/tcl8.3
make
make install

2. If you need our version of Tk:

cd $CCPNMR_TOP_DIR/tk8.3
Unpack tar file: 
tar xvzf tk8.3.4.tar.gz
(or "gunzip" followed by "tar xvf" if your tar does not understand "z") Compile code: 
cd tk8.3.4/unix
./configure --prefix=$CCPNMR_TOP_DIR/tk8.3 --with-tcl=$CCPNMR_TOP_DIR/tcl8.3/lib
(or your own "with-tcl" argument if you are not using our Tcl) 
make
make install

3. If you need our version of Python:

cd $CCPNMR_TOP_DIR/python2.2
Unpack tar file: 
tar xvzf Python-2.2.3.tgz
(or "gunzip" followed by "tar xvf" if your tar does not understand "z") Edit setup: 
cd Python-2.2.3
copy Modules/Setup.dist to Modules/Setup edit Modules/Setup: below the line that says "# The _tkinter module." edit the lines as instructed Compile code: (In the Python-2.2.3 directory.) 
./configure --prefix=$CCPNMR_TOP_DIR/python2.2
make
make install

4. If you need our version of Mesa (so in particular want OpenGL capability):

cd $CCPNMR_TOP_DIR/mesa6.0
Unpack tar files: 
tar xvzf MesaLib-6.0.tar.gz
tar xvzf MesaDemos-6.0.tar.gz
(or "gunzip" followed by "tar xvf" if your tar does not understand "z")

Compile code: 
cd Mesa-6.0
make
(shows possible make choices: e.g. linux-x86 for Linux on PCs) 
make <your choice of architecture>
cd ..
ln -s Mesa-6.0/include include
ln -s Mesa-6.0/lib lib

5. Compile the Analysis code:

cd $CCPNMR_TOP_DIR/ccpnmr1.0/c
cp environment_default.txt environment.txt
edit environment.txt changing the variables as required (specifically IGNORE_GL_FLAG, X11_DIR, TCL_DIR, TK_DIR, PYTHON_DIR, and GL_DIR if IGNORE_GL_FLAG is blank) 
make
make links

6. Create the bin directory:

cd $CCPNMR_TOP_DIR
mkdir bin
cd bin
create file "analysis" with one line in it: 
python -i -O $CCPNMR_TOP_DIR/ccpnmr1.0/python/ccpnmr/analysis/AnalysisGui.py $1 $2 $3

create file "formatConverter" with one line in it: 

python -O $CCPNMR_TOP_DIR/ccpnmr1.0/python/ccpnmr/format/gui/FormatConverter.py $1
Then 
chmod 755 analysis formatConverter
and if you use the distributed python for running the applications: 
ln -s ../python2.2/bin/python python
