This popup shows the individual constraint items of a selected constraint list.

Constraint Sets: <list>

Gives a list of the constraint sets within the current project. Allows the user to select one of these from which a constraint list can then be selected.

Constraint List: <list>

Allows selection of a constraint list from those present in the currently selected constraint set. The constraint items of the selected list are then displayed below in the main table.

Structure: <list>

Sets the structure on which constraint connections may be shown.

- Constraints -

This table displays the constraints and the constraint items for the currently selected list.

A constraint carries information about the value of the constraint (target value, limits etc.), but each of its constraint items represents one of possibly many alternative situations to which the constraint is applied. For example a distance constraint of so many Angstroms may be applied ambiguously to two different pairs of atoms. Each alternative pair of constrained atoms (specified via resonances) represents one resonance item.

The individual rows of the table correspond to individual constraint items. However, the first item of a constraint is coloured differently from the others and carries the information for the constraint itself.

| # |

The number of the constraint item in the form constraintId:itemId. For example 100:5 would represent the fifth (ambiguous) item of the hundredth constraint.

| Resonances |

Indicates which resonances are being constrained by a constraint item.

| Value | Upper Limit | Lower Limit |

The target values and limits for the constraint. Note that this information is only displayed for the first item of each constraint.

| Peaks |

The number of peaks used to generate this constraint item. Double clicking this column will display these peaks in a separate table. Opens SelectedPeaksPopup.

[ Delete Items ]

Delete the currently selected constraint items. If all of the items within a constraint are deleted, then the whole constraint will be deleted.

[ Delete Constraints ]

Delete all of the constraints pertaining to the currently selected items. Note that this will delete unselected items if at least one of the other items in its parent constraint is selected.

[ View Peaks ]

Displays the peaks associated with the currently selected constraint(s). The peaks are listed in a separate popup table. Opens SelectedPeaksPopup.

[ View Violations ]

This option is not yet functional.

[ View Selected On Strcture ]

Displays connections representing the currently highlighted constraints on the current structure using ViewStructurePopup.

[ Setup Resonances ]

This option generates new resonances for any of the constraints within the current constraint list that are not linked (at either end) to an assignable resonance. This option is useful for making resonances from newly imported constraint lists which don't yet pertain to any assignments.

[ Export ARIA List ]

Opens a FileSelect popup to select a file name. An ARIA constraints table (.tbl) format file representing the current list of constraints will then be written to the selected file name.

[ Delete List ]

Deletes all of the constraints belonging to the list currently being displayed in the main table.
