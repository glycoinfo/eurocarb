This popup window shows the probabilities of a spin system being each of several residue types. The probability score is generated by comparing the chemical shifts of the resonances in the spin system with the known chemical shift distributions for residues from the BMRB data set.

Spin System: Serial <serialNumber> Assignment: <residueName>

This label shows the serial number of the current spin system and indicates whether it is assigned to any specific type or residue in a molecule.

Shift List: <list>

This option sets which shift list (from the current project) to use to obtain chemical shifts for the resonances in the current spin system.

Chain: <list>

This option sets which molecular chain the spin system corresponds to. The residues in this molecule govern the types of residue probabilities are calculated for.

- Resonances -

This table lists all of the resonances within the current shift list that are considered when calculating residue type probibilities.

| Isotope |

The the type of isotope of the resonance that gives rise to a chemical shift.

| Name |

The name of the resonance: If a resonance is assigned to a specific atom type then this name indicates the type of atom or atoms to which the resonance is assigned.

| Shift Value |

The chemical shift of a resonance within the selected shift list.

| Shift Error |

The standard deveation of the resonance's chemical shift. Note that that this is calculated considering the individual weighings of spectrum dimensions; e.g. which may be deminished if peaks are especially imprecise in a given dimension.

[ Remove From Spin System]

Removes the currently selected resonance from the current spin system. Note that this does not affect the atom type to which the resonance may be assigned.

[ Show Peaks ]

Displays all of the peaks which are linked to the currently selected resonance using SelectedPeaksPopup.

- Type Scores -

This table shows the probabilities for the current spin system being each of the residue types represented in the selected molecule chain. The rows of the table are colour coded according to the likelyhood of each residue type. The colours indicate the strength of probability as follows: Green - likely; Yellow - possible but not convincing; Orange - unlikely; Red - very unlikely.

| Rank |

The rank of the possibility, with the most likely residue type having th elowest rank.

| CcpCode |

This column shows the CcpCode (similar to three letter code) for the residue type that a probability is calculated for.

| % Probability|

The percentage probability for a spin system being a given residue type, relative to the other possible residue types for the currently selected chain.

[ Assign Spin System Type ]

Assign the current spin system to the currently selected residue type (lower) table. Note that this assignment does not affect any atom type assignments a residue may have.
