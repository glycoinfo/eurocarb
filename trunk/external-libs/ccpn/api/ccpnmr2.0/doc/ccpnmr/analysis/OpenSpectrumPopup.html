This popup window is used to select various parameter/data files for spectra. When a spectrum file is opened a spectrum object (data source in the Data Model) is created in the current project. These spectra may be associated with new or existing NMR experiments.

- File Selection -

This section of the popup allows the user to navigate file systems and open a directory from which multiple selected spectra may be selected for loading. The type of files that are selected in the table depends on the type of file format set . For example to load spectra that have been processed with Azara, the parameter files are selected. Whereas for Felix data the spectrum data files (which contain header information) themselves are selected. In any instance the selected files specify how spectral data should be interpreted and what the location of the data is.

File format: <list>

Allows the user to specify what format selected spectrum/parameter files use. Available options include Azara, Felix, NMRPipe (NMRView) and UCSF formats.

[] verification dialogs:

If set to ON, when the selected spectra are loaded their parameters any any defaults are assumed to be correct. Analysis will attempt to load the spectra with minimal user input.

If set to OFF, the user will be prompted to confirm the file and referencing parameters that have been read for each spectrum. Here the user has the opportunity to edit spectrum parameters prior to leading.

- Spectra to open -

This table lists the spectra that may be created from the selected files. Each selected file gives rise to a spectrum for a specified experiment.

| Experiment |

The name of the experiment that generated the spectrum. This may be edited by double clicking. If there is an existing NMR experiment with the name, the loaded spectrum will be assumed derive from the that experiment. Otherwise a new experiment will be created for the loaded spectrum (and the name may be changes later).

| Spectrum |

The name of the spectrum to be loaded, set by double clicking. This is the identifier for a spectrum within its experiment. The value defaults to the file name selected for the spectrum.

| Shift List |

This column displays which chemical shift list the experiment and spectrum will be associated with. The shift list can be set by double clicking. The user can select from existing lists and there is the option to create a new shift list for the spectrum "".

[ Open spectra ]

Reads the selected spectrum/parameter files to load spectra into the current project, according to the experiment selection specified in the lower table and activates EditSpectrumFilePopup.
