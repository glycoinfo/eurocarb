<dim1> <dim2> <dim3> in <windowName>

Goto an equivalent position in a named window. The target windows include all those that have at least one axis in common with the current (source) window. Any equivalent axis in the target window will be cantered on the position corresponding to the cursor position in the current window. Where a target window had more than one strip region, this function will affect only the active strip. This option is commonly used to find Z planes in windows with orthogonal axis mappings from the current X-Y position.

All window positions

Goto all of the above listed window locations, i.e. find the current position in all possible windows

Transpose <aixs_type> <axes> <position>

Goto a transpose position in a window wich has two axes of the same type. For example at position x=4.3, y=7.2 in a 1H, 1H 2D window the transpose position will be x=7.2, y=4.3. The funtion is not restricted to 2D experiments.
