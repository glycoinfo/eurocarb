This popup allows the user to define ExpGraphs, that determine the magnetisation transfer network between the AtomSites and order in which it is traversed.

NmrExpPrototype <list>

Select the NmrExpPrototype you want to view/edit

- ExpGraphs -

Create/select the ExpGraph. An ExpGraph defines the magnetisation transfer steps in the experiment, and the order of the steps. Most experiments have only one ExpGraph, but you may need more than one where there are several distinct magnetisation pathways within a single experiment.

| PeakSign |

Whether the peaks arising from this ExpGraph have positive or negative (relative) sign. Generally left empty unless there are several ExpGraphs that produce peaks of different sign.

[ Create New ]

Add a new ExpGraph.

[ Delete ]

Deletes the currently selected ExpGraph.

- ExpSteps -

Defines the order in which the magnetisation passes through the different ExpMeasurements for the curerntly selected ExpGraph

| StepNumber |

Step number for the ExpStep

| Measurement |

ExpMeasurement corresponding to the ExpStep

[ Create New ]

Add a new ExpStep with the next available stepNumber using the currently selected ExpMeasurement. The value can be edited afterwards.

[ Re-number]

Renumber the ExpSteps (without changing their order) so the numbers are consecutive starting from one.

[ Delete ]

Deletes the currently selected ExpStep.

- ExpTransfers -

Defines the magnetisation transfer steps that connect the AtomSites for the currently selected ExpGraph.

| transfer Type |

The type of transfer between the AtomSites. Note that NOESY includes ROESY transfers. onebond transfer is generally by J cpupling, but may use otehr mechanisms. Jmultibond means J coupling transfer over more than one bond, while Jcoupling means transfer over any number of bonds. Jmultibond should only be used where one-bond coupling is possible but suppressed. I.e. use Jcoupling rather than Jmultibond for H-H couplings.

| AtomSite1 |

| AtomSite2 |

The two AtomSites involved in the transfer - the sites must be different. They are always shown in sorted order.

| transferToSelf |

Does the magnetisation have to be transferred to a different AtomSite, or may it remain where it started (corresponding to forming a diagonal peak). transferToSelf can only be True if the two AtomSites have the same isotopeCode.

[ Create New ]

Add a new ExpTransfer with default values. The transfer can be edited afterwards.

[ Delete ]

Deletes the currently selected ExpTransfer.
