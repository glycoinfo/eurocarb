This popup displays tables of information for the sets of constraints present within the current project. A constraint set is a means of grouping together constraints (distance, dihedral etc.), constraint violations and the assignments present at the time that the constraints were generated.

- Constraint Sets -

This table lists the constraint sets within the current project. Clicking on an occupied row in the table will fill the lower tables of the popup with details of the constraint and violation lists within the selected constraint set. Each constraint set corresponds to one seperately stored XML file (NmrConstraints package). This means that each constraint set is loaded separately (and only when required) from other sets and the main NMR data.

Columns indicating "Fixed" items indicate how many of these various assignment objects are used within the constraint set. The "Fixed" data items are a snapshot of the regular assignment items. In this manner assignment of the project may subsequently be modified, but previously created constraints remain unaffected. Also, the exact assignment status of the project at the time the constraint set was created is recorded.

[ Import ]

This option is not yet functional.

[ Delete ]

Removes the currently selected constraint set from the current project. Note that this will NOT remove the corresponding NmrConstrants package XML file from disk.

- Constraint Lists

This table displays information about the lists of constraints contained within the currently selected constraint set.

| Type |

This column indicates what the constraints in the list are constraining, commonly distance or dihedral angle.

| Experiments |

The experiments that give rise to the peaks from which the constraints in the list were generated.

| Unit |

The units of measurement for the constraint. E.g. Angstroms, radians etc.

[ View Constraints ]

Display the individual constraint items of the currently/primary selected list in a popup. Opens BrowseConstraintsPopup.

[ Create List ]

This option is not yet functional.

[ Delete Lists ]

Delete the currently selected lists and their constituent constraints.

- Violation Lists -

A table which displays basic information about the lists of violations within the currently selected constraint set.

| Structures |

Indicates which structures the violations are calculated for.

[ View Violations]

Display the individual violation items of the currently/primary selected violation lists in a popup table. Opens BrowseViolationsPopup.

[ Delete Lists ]

Deleted the currently selected violation lists and their violation items.
